## Big O Notation ‡∏ó‡∏±‡πà‡∏ß‡πÑ‡∏õ

### O(1) - ‡πÄ‡∏ß‡∏•‡∏≤‡∏Ñ‡∏á‡∏ó‡∏µ‡πà
‡πÄ‡∏ß‡∏•‡∏≤‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÉ‡∏ô‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£‡πÑ‡∏°‡πà‡∏Ç‡∏∂‡πâ‡∏ô‡∏≠‡∏¢‡∏π‡πà‡∏Å‡∏±‡∏ö‡∏Ç‡∏ô‡∏≤‡∏î‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•

### O(n) - ‡πÄ‡∏ß‡∏•‡∏≤‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô
‡πÄ‡∏ß‡∏•‡∏≤‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÉ‡∏ô‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£‡∏à‡∏∞‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡∏ï‡∏≤‡∏°‡∏Ç‡∏ô‡∏≤‡∏î‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•

    }

### O(log n) - ‡πÄ‡∏ß‡∏•‡∏≤‡∏•‡∏≠‡∏Å‡∏≤‡∏£‡∏¥‡∏ó‡∏∂‡∏°
‡πÄ‡∏ß‡∏•‡∏≤‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÉ‡∏ô‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£‡∏à‡∏∞‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÅ‡∏ö‡∏ö‡∏•‡∏≠‡∏Å‡∏≤‡∏£‡∏¥‡∏ó‡∏∂‡∏° ‡∏ï‡∏≤‡∏°‡∏Ç‡∏ô‡∏≤‡∏î‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏• 

```go
package main

import "fmt"

func main() {
    slice := []int{1, 3, 5, 7, 9}
    target := 7
    left := 0
    right := len(slice) - 1
    for left <= right { // O(log n)
        mid := (left + right) / 2
        if slice[mid] == target {
            fmt.Println("Found!")
            break
        } else if slice[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
}
```

### O(n^2) - ‡πÄ‡∏ß‡∏•‡∏≤‡∏Å‡∏≥‡∏•‡∏±‡∏á‡∏™‡∏≠‡∏á
‡πÄ‡∏ß‡∏•‡∏≤‡∏ó‡∏µ‡πà‡πÉ‡∏ä‡πâ‡πÉ‡∏ô‡∏Å‡∏≤‡∏£‡∏î‡∏≥‡πÄ‡∏ô‡∏¥‡∏ô‡∏Å‡∏≤‡∏£‡∏à‡∏∞‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏Ç‡∏∂‡πâ‡∏ô‡πÅ‡∏ö‡∏ö‡∏Å‡∏≥‡∏•‡∏±‡∏á‡∏™‡∏≠‡∏á ‡∏ï‡∏≤‡∏°‡∏Ç‡∏ô‡∏≤‡∏î‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•

```go
package main

import "fmt"

func main() {
    slice := []int{1, 2, 3, 4, 5}
    for i := 0; i < len(slice); i++ { // O(n)
        for j := 0; j < len(slice); j++ { // O(n)
            fmt.Println(slice[i], slice[j])
        }
    }
}
```

### O(n log n) - ‡πÄ‡∏ß‡∏•‡∏≤‡∏•‡πá‡∏≠‡∏Å‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô

```go
package main

import "fmt"

func main() {
    slice := []int{5, 2, 4, 6, 1, 3}
    for i := 0; i < len(slice)-1; i++ { // O(n)
        for j := i + 1; j < len(slice); j++ { // O(log n)
            if slice[i] > slice[j] {
                temp := slice[i]
                slice[i] = slice[j]
                slice[j] = temp
            }
        }
    }
    fmt.Println(slice) 
}
```

* **O(2^n) - ‡πÄ‡∏ß‡∏•‡∏≤‡πÄ‡∏•‡∏Ç‡∏ä‡∏µ‡πâ‡∏Å‡∏≥‡∏•‡∏±‡∏á:** 

```go

func main() {
    n := 5
    result := 0
    for i := 0; i < n; i++ { // O(2^n)
        for j := 0; j < n; j++ {
            result += 1
        }
    }
    fmt.Println(result)
}
```
